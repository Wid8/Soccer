---

## התקנה על Railway

### שלב 1 — צור חשבון Railway
1. כנס ל־ https://railway.app
2. לחץ "Login with GitHub" (תצטרך חשבון GitHub חינמי)

### שלב 2 — העלה את הקוד ל-GitHub
1. כנס ל- https://github.com וצור repository חדש בשם "football-bot"
2. העלה את כל הקבצים (index.js, package.json)

### שלב 3 — צור פרויקט ב-Railway
1. ב-Railway לחץ "New Project"
2. בחר "Deploy from GitHub repo"
3. בחר את ה-repository שיצרת
4. Railway יתחיל להריץ את הבוט אוטומטית

### שלב 4 — חבר וואטסאפ (פעם אחת בלבד)
1. ב-Railway לחץ על הפרויקט → "Deployments" → "View Logs"
2. תראה QR code בלוגים
3. פתח וואטסאפ בטלפון → שלוש נקודות → מכשירים מקושרים → קשר מכשיר
4. סרוק את ה-QR
5. זהו! הבוט מחובר ועובד

## פקודות ניהול

### עצירת הודעות (אם סגרת חובות ידנית)
```
node index.js stop
```

### בדיקת סטטוס נוכחי
```
node index.js status
```

### סימון תשלום ידני
```
node index.js paid "שם"
```

### בדיקת הודעת שלישי (לטסט)
```
node index.js test-tuesday
```

### בדיקת תזכורת (לטסט)
```
node index.js test-reminder
```
